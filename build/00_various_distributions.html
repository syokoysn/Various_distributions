0. 様々な確率分布¶

確率分布には，正規分布， 一様分布など様々な確率分布が存在する.
ここでは, それらの分布がどのような分布か主にScipyを用いて確かめることを目的とする．

0.1 扱う確率分布¶

以下の確率分布を扱う

  • 一様分布 (uniform distribution)

  • 指数分布 (exponential distribution)

  • Cauchy分布 (Cauchy distribution)

0.2 各分布の見た目¶

各確率分布の内容を確認する前に，大まかなにどんな分布かを想像できるよう，ここで各確率分布を図示する．

[14]:

import plotly.express as  px
import plotly.graph_objects as go

0.2.1 一様分布¶

[15]:

from scipy.stats import uniform

[16]:

uni = uniform(100, 50)# [100, 100+50 ]の一様分布作成
x = 10000
y = uni.rvs(size = x)
fig = go.Figure(data=[go.Histogram(x=  y)])
